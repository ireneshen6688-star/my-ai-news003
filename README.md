# My AI News

> 智能新闻订阅工具 - 让 AI 成为你的私人新闻编辑

## 项目简介

My AI News 是一款基于 AI 的个性化新闻订阅工具，用户可以自定义关键词，系统会定时抓取相关新闻并通过邮件推送。

## 核心功能

- ✅ Google 账号登录
- ✅ 自定义关键词订阅
- ✅ 定时邮件推送（每天/每周/每月）
- ✅ AI 智能摘要
- ✅ 任务管理（开启/关闭）
- ✅ 多语言支持（英文/中文）

## 技术栈

- 前端：Next.js + TypeScript + Tailwind CSS
- 后端：Cloudflare Workers
- 数据库：Cloudflare D1
- AI：OpenAI API
- 邮件：Resend
